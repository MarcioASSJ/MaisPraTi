Primeiro deve conferir se o node esta corretamente instalado, para isso, abra o terminal e digite "node -v", caso esteja instalado, aparecerá a versão utilizada, caso não apareça, baixe e instale o Node de um site oficial. Em seguida, é necessária a instalação do prompt-sync no Visual Studio Code, abra o programa e dentro e abra o terminando teclando CTRL+SHIFT+' e digite "npm install prompt-sync", após a instalação, será possivel executar código. Selecione todos os arquivos .js enviado, clique com o botão direito e selecione "abrir com code", feito isso basta executar o mesmo teclando CTRL+ALT+N ou no terminal ja aberto executar o comando "node Exercise1-15.js",nessa segunda opção deve ficar atento ao diretório em que o Terminal está, pois deve executar o comando no mesmo diretório onde deixou o arquivo baixado. Todos os exercícios estão no mesmo arquivo basta  conferir no terminal qual o exercício que está rodando no momento e seguir as orientações do programa.
